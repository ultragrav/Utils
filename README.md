# Utils
A collection of utilities for Bukkit/Spigot developers

## Installation
### Maven:
```xml
<repository>
    <id>UltraGrav</id>
    <url>https://mvn.ultragrav.net</url>
</repository>

<dependency>
    <groupId>net.ultragrav</groupId>
    <artifactId>Utils</artifactId>
    <version>1.0.3</version>
</dependency>
```
